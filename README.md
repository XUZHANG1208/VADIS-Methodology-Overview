# VADIS-Methodology-Overview
This repository contains datasets, codes and outputs from all previous VADIS projects. 

The folder "VADIS_world_englishes" containts the metadata to the project: (Szmrecsanyi, Grafmiller & Rosseel 2019) and later (Szmrecsanyi & Grafmiller 2023).

The folder "VADIS_Probabilistic_Indigenization" corresponds to the project: (Tamaredo et al. 2020)

The folder "VADIS_English_FTR" corresponds to the project: (Bartels & Szmrecsanyi 2025)

The folder "VADIS_Epicentral_Influence" corresponds to the project: (La Peruta 2022)

The folder "VADIS_Chinese_Theme_Recipient" corresponds to the project: (Li, Szmrecsanyi & Zhang 2024)

The folder "VADIS_Register" corresponds to the project: (Zhang & Szmrecsanyi 2024)
